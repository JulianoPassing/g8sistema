# Ícones PWA - G8 Sistema

Para melhor suporte offline e instalação como PWA, adicione os ícones nesta pasta:

- **icon-192.png** - 192x192 pixels
- **icon-512.png** - 512x512 pixels

Você pode gerar os ícones a partir do logo em: https://i.imgur.com/WveVVY5.png

Depois de adicionar os arquivos, o manifest.json usará os caminhos locais automaticamente.
