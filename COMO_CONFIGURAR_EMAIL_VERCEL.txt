🎯 PASSO A PASSO SIMPLIFICADO
────────────────────────────────────────────────────────────────

📧 PASSO 1: GERAR SENHA DO GMAIL
────────────────────────────────────────────────────────────────

1. Abra: https://myaccount.google.com/apppasswords

2. Faça login no Gmail

3. Selecione: "Outro (nome personalizado)"

4. Digite: Sistema G8 Vercel

5. Clique em: GERAR

6. COPIE a senha de 16 dígitos
   Exemplo: abcd efgh ijkl mnop
   
   ⚠️  GUARDE ESSA SENHA - você vai usar no próximo passo!


🌐 PASSO 2: CONFIGURAR NO VERCEL
────────────────────────────────────────────────────────────────

1. Acesse: https://vercel.com/dashboard

2. Clique no projeto: g8sistema

3. Clique em: Settings (⚙️)

4. No menu lateral: Environment Variables

5. Adicione 3 variáveis:


   ┌─────────────────────────────────────────────────────────┐
   │ Variável 1:                                             │
   ├─────────────────────────────────────────────────────────┤
   │ Key:   EMAIL_USER                                       │
   │ Value: seuemail@gmail.com                               │
   │ Environment: ✅ Production ✅ Preview ✅ Development     │
   └─────────────────────────────────────────────────────────┘

   ┌─────────────────────────────────────────────────────────┐
   │ Variável 2:                                             │
   ├─────────────────────────────────────────────────────────┤
   │ Key:   EMAIL_PASS                                       │
   │ Value: abcd efgh ijkl mnop (senha gerada no passo 1)   │
   │ Environment: ✅ Production ✅ Preview ✅ Development     │
   └─────────────────────────────────────────────────────────┘

   ┌─────────────────────────────────────────────────────────┐
   │ Variável 3:                                             │
   ├─────────────────────────────────────────────────────────┤
   │ Key:   EMAIL_TO                                         │
   │ Value: destinatario@gmail.com (quem recebe os e-mails) │
   │ Environment: ✅ Production ✅ Preview ✅ Development     │
   └─────────────────────────────────────────────────────────┘


   Como adicionar cada uma:
   → Clique em "Add New"
   → Preencha Key e Value
   → Marque TODOS os ambientes
   → Clique em "Save"


🚀 PASSO 3: FAZER REDEPLOY
────────────────────────────────────────────────────────────────

Escolha uma opção:

OPÇÃO A (Recomendada):
   → Faça um commit no Git
   → Dê push para GitHub
   → Vercel faz deploy automaticamente

OPÇÃO B (Manual):
   → Dashboard Vercel → Deployments
   → Clique nos "..." do último deploy
   → Clique em "Redeploy"


✅ PASSO 4: TESTAR
────────────────────────────────────────────────────────────────

1. Aguarde o deploy finalizar (1-2 minutos)

2. Acesse seu sistema no Vercel

3. Crie um pedido novo (teste)

4. Aguarde 10-30 segundos

5. Verifique seu e-mail
   ⚠️  DICA: Se não aparecer, olhe na pasta SPAM!


🎉 PRONTO!
────────────────────────────────────────────────────────────────

Agora você receberá e-mails automaticamente sempre que
um novo pedido for criado no sistema!


❓ PROBLEMAS?
────────────────────────────────────────────────────────────────

❌ "E-mail não chega"
   → Verifique a pasta SPAM
   → Aguarde 1 minuto
   → Confirme que fez o redeploy

❌ "Invalid login"
   → Gere uma NOVA senha de aplicativo
   → Delete EMAIL_PASS antiga no Vercel
   → Adicione a nova senha
   → Faça redeploy novamente

❌ "Variáveis não funcionam"
   → Confirme que marcou "Production"
   → Faça redeploy
   → Aguarde deploy completar
